This is a small starter project for an educational platform built with Vite + React + TypeScript and Tailwind CSS.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Open http://localhost:5173

What this scaffold includes

- Vite + React + TypeScript
- Tailwind CSS configured
- Folder structure: `src/components`, `src/pages`, `src/layouts`, `src/data`, `src/types`
